HTML

HyperText Markup Language

Linguagem de Marcação de Hipertexto

Não é uma linguagem de programação

HTML é formado por
TAGS

Tags de texto

tag  P (paragraph)

                            <p>este é um parágrafo</p>

tag  strong (negrito)

                            <strong>texto em negrito</strong>

tag  em (emphasis)

                            <em>texto em itálico</em>

tag  hr (horizontal ruler)

                            <hr>
Não necessita de fechamento

tag  br (line break)

                            <br>
Não necessita de fechamento

tag  h1 (heading 1)

                            <h1>título nível 1</h1>

tags  h2 a h6 (heading 2 a 6)

                            <h2>título nível 2</h2>
                            ...
                            <h6>título nível 6</h6>

tags  ul (unordered list - lista desordenada) e li (list item - item de lista) 

                            <ul>
                              <li>item 1</li>
                              <li>item 2</li>
                            </ul>

Atenção:  Identação 

                                <ul>
                                <li>item 1</li>
                                <li>
                                <ul>
                                <li>item 1</li>
                                <li>item 2</li>
                                </ul>
                                </li>
                                </ul>
                                
                            
                                <ul>
                                |___<li>item 1</li>
                                |___<li>
                                |___|___<ul>
                                |___|___|___<li>item 1</li>
                                |___|___|___<li>item 2</li>
                                |___|___</ul>
                                |___</li>
                                </ul>

Atenção:  Algo quebrado?

Você tem certeza que fechou corretamente todas as tags?

                            <ul>
                              <li>item 1</li>
                              <li>
                                <ul>
                                  <li>item 1<li>
                                  <li>item 2</li>
                                </ul>
                              </li>
                            </ul>

tag  ol (ordered list - lista ordenada) 

                            <ol>
                              <li>item 1</li>
                              <li>item 2</li>
                            </ol>

tags de formulário

tag  input 

                            <input>

tag  textarea 

                            <textarea>texto</textarea>

tags  select e option 

                            <select>
                              <option>opção 1</option>
                              <option>opção 2</option>
                            </select>

tag  button (botão)

                            <button>salvar</button>

Tags de tabela

tags  table , tr , th e td 

                            <table>
                              <tr>
                                <th>Nome</th>
                                <th>Idade</th>
                              </tr>
                              <tr>
                                <td>Maria</td>
                                <td>30</td>
                              </tr>
                              <tr>
                                <td>João</td>
                                <td>40</td>
                              </tr>
                            </table>

Tags de estrutura

tags  html , head , title e body 

                            <html>
                              <head>
                                <title>Título da página</title>
                              </head>
                              <body>
                                <p>conteúdo</p>
                              </body>
                            </html>

tags  div e span 

                            <div></div>
                            <span></span>

Atributos 

                                <tag atributo="valor">conteúdo</tag>

tag  A (anchor)

                            <a href="google.com">este é um link</a>

tag  img (imagem)

                            <img src="img.jpg" alt="minha imagem">

Comentário
